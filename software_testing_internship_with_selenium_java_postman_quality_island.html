Introduction

As part of my transition into the IT industry, I have completed a comprehensive training and internship program designed to equip future testers with both theoretical foundations and practical, commercial experience. Delivered by Quality Island and mentored by industry veteran Tomasz Stelmach, the program offers real-world exposure to software testing within a professional Agile environment.


Key Highlights

This internship turned knowledge into action. Working on a commercial-grade web application, I applied test automation and API testing skills in a real project environment. Using tools like Selenium, Java, Postman, and AI technologies, I’ve built the kind of experience today’s QA teams expect from a new hire.

  • Hands-On Commercial Experience: I participated in manual and automated testing of a live, production-grade web application for an actual business client. My responsibilities included writing test cases and regression test scripts, executing both manual and Selenium-based automated tests, reporting bugs, and working with documentation and test reports—all in a collaborative, real-world setting.
  • Test Automation Technologies: Gained practical skills in Selenium WebDriver, using Java and TestNG, applying Page Object Pattern, and working with tools such as Maven, Git, and Jenkins to build maintainable, scalable test suites.
  • API Testing & Automation: Developed proficiency in Postman for REST API testing, including automated scripts, authorization handling, and integration with CI pipelines using Newman.
  • AI in Software Testing: Explored the use of AI-powered testing tools like GitHub Copilot, Testim, and Applitools to accelerate and enhance test creation, bug detection, and test optimization. Also practiced using ChatGPT to generate code snippets, test data, and CI/CD prompts.
  • Development & Tooling Ecosystem: Learned essential tools for testers including JIRA, Zephyr Scale, DevTools, and GIT, which support end-to-end test case management, bug tracking, and version control.
  • Complementary Skills: Participated in themed workshops covering mobile testing (Android & iOS), performance testing (e.g., K6, JMeter), accessibility testing (WCAG 2.2), SQL basics for database validation, and RPA tools (UiPath, Blue Prism, Automation Anywhere).
  • Certified Foundation: Completed a dedicated module aligned with ISTQB Certified Tester Foundation Level, establishing a solid base in test theory, methodology, and structured test processes.

Why It Matters

This program bridges the gap between training and employment. Beyond theoretical know-how, it gave me the commercial testing experience that hiring managers look for—proving I can apply my skills in real project environments, deliver results as part of a testing team, and contribute to product quality from day one.
