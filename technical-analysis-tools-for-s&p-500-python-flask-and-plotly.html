Flask Application

Market Bias
Market Bias is a web application built on Python and Flask, designed to provide users with in-depth historical data and analysis of the S&P 500 index.

Leveraging the power of Pandas and Plotly, the application connects to the Yahoo Finance API via Pandas Datareader, applying various technical analysis tools to offer valuable insights. A standout feature is its interactive visualizations, which allow users to easily view market bias and explore different trends. Additionally, the application offers a simple yet profitable trading strategy known as Dual Momentum. The website also includes a dynamically updated table of various stock indices, grouped by their one-year rate of change, sourced daily from the Yahoo Finance API.

Key Features:

  • Data Retrieval and Analysis

    Yahoo Finance API Integration: Utilizes Pandas Datareader to fetch historical data from Yahoo Finance. The data is then processed and analyzed using Pandas, offering various technical analysis tools to provide valuable insights into the S&P 500 index.

    Daily Updates: The application automatically updates a table of various stock indices, grouped by their one-year rate of change, ensuring users have access to the most recent data.


  • Interactive Visualizations

    Plotly Integration: Implements Plotly to create interactive and visually appealing charts. Users can explore market trends, biases, and other technical indicators through dynamic visualizations.


  • Performance Optimization

    HTML and CSS Minification: Optimizes loading times by minifying HTML and CSS files, removing unnecessary characters to reduce file sizes and improve website performance.

    Background Email Sending: Moves the email sending function to a background thread, significantly reducing delays during request handling and enhancing the overall user experience.


  • Responsive and Modern Design

    CSS Flexbox: Employs CSS Flexbox to simplify complex layout structures, making it easy to arrange and align items within a container. Flexbox's compatibility with modern browsers ensures a responsive and user-friendly design.

    Font Awesome Icons: Enhances the design and functionality with customizable icons from Font Awesome, allowing for a more engaging user interface.


  • Security and Best Practices

    Environment Variables: Stores sensitive information such as API keys and database passwords in environment variables to enhance security.

    Flask Blueprints: Uses Flask blueprints for modular organization of the application, improving code maintainability and scalability.

    Application Factory Pattern: Separates the creation of the application object from the rest of the codebase, allowing for easy configuration and management.
